# Overview

The AI Based Daily Habit Tracker is designed to solve a common problem faced by students maintaining consistency in daily routines like studying, exercising and managing time.

## This project not only tracks habits but also uses Machine Learning to -

Predict whether a habit will be completed

Provide confidence scores

Offer basic behavioral insights

# Features

Add and manage daily habits

Mark habits as completed or missed

Track streaks for motivation

View progress with completion percentage

Get daily reminders 
Basic analytics

AI prediction using Machine Learning

# AIML Concepts Used

Supervised Learning

Logistic Regression

Feature Engineering 

Prediction & probability scoring

# How to Use

Run the program
Choose options from the menu -
1. Add Habit
2. Mark Habit
3. View Progress
4. Reminders
5. Analytics
6. Predict Habit 
7. Exit

# Example Workflow

Add habits like:

Study 2 hours

Exercise

Mark them daily (Completed / Missed)

After 5+ days:

Use AI Prediction feature

AI Prediction:

You are likely to COMPLETE 'Study 2 hours' today 

Confidence: 82.45%
